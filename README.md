# Zoom Chapter Splitter

AI-powered desktop application for splitting Zoom recordings into digestible chapters.

## Features

- Import Zoom recordings (MP4) and transcripts (VTT)
- AI-powered chapter detection using OpenAI or Anthropic
- Visual timeline with chapter markers
- Adjustable timecodes with padding controls
- Title card generation with customization
- Fade in/out effects
- Batch export with progress tracking

## Development

```bash
# Install dependencies
npm install

# Run in development mode
npm run dev

# Build for production
npm run build

# Package for macOS
npm run package
```

## Requirements

- macOS 12+
- Node.js 18+
- OpenAI or Anthropic API key
